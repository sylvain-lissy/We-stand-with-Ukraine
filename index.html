We-stand-with-Ukraine

Just a little CSS to add to your website to stand with Ukraine

How-to use

First, you have to add the class we-stand-with-ukraine to a <div> :

  1. <div title="We stand with Ukraine" class="we-stand-with-ukraine"></div>

And then, copy this code in the <head>...</head> section of your page : 

  1. <link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/sylvain-lissy/we-stand-with-ukraine/we-stand-with-ukraine.css">

Or, you can also add the CSS diretly into your stylesheet or HTML :

  1. .we-stand-with-ukraine {
  2.         left: -80px;
  3.         bottom: 20px;
  4.         transform: rotate(45deg);
  5.         background: linear-gradient(-180deg, rgb(0, 91, 187) 50%, rgb(255, 213, 0) 50%);
  6.         width: 300px;
  7.         height: 84px;
  8.         position: fixed;
  9.         z-index: 999;
  10.     }
